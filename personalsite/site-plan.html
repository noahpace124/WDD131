Overview

Purpose

This website will allow the user to collect random jokes from a text file of jokes. It will also allow the user to submit jokes to be added to the text file for future users.

Audience

This website is for anybody seeking a good laugh or that want their jokes to get spread.

Dynamic elements

The dynamic elements of the site would be the importing and exporting of a joke.txt file, collecting a random joke from the list when a button is pressed, and the submition of a joke to be added.
